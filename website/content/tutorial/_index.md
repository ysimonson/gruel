Welcome to the Rue tutorial! This guide will teach you the language step by step, from your first program to implementing algorithms.

Rue is an early-stage compiled language exploring memory safety. It's not ready for real projects yet, but if you're curious about where it's headed, this tutorial will show you what we have so far.
