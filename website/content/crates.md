The Gruel compiler is organized as a Cargo workspace. The API documentation below is generated by `rustdoc` from the source of each library crate.

These docs describe internal compiler APIs and are primarily useful for contributors. For the language itself, see the [Specification](@/spec/_index.md) and [Tutorial](@/tutorial/_index.md).
