Welcome to Learn Gruel. This guide walks through the language step by step, from your first program to implementing algorithms. Reference material for intrinsics and design decisions lives under [References](@/learn/references/_index.md).

Gruel is an early-stage compiled language exploring memory safety. It's not ready for real projects yet, but if you're curious about where it's headed, this guide will show you what we have so far.
