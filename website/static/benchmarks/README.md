# Benchmark Data

This directory contains benchmark history data for the Gruel compiler performance dashboard.

## Files

- `history.json` - Historical benchmark results (auto-generated by CI)

## Data Format

The `history.json` file contains an array of benchmark runs:

```json
{
  "version": 1,
  "runs": [
    {
      "timestamp": "2025-12-27T10:30:00Z",
      "commit": "abc123def",
      "host": {"os": "darwin", "arch": "arm64"},
      "iterations": 5,
      "benchmarks": [
        {"name": "many_functions", "mean_ms": 10.5, "std_ms": 0.5, "iterations": 5}
      ]
    }
  ]
}
```

## Updating History

History is updated automatically by CI on each commit to trunk. To run manually:

```bash
# From repository root
./bench.sh

# Or to just save results without updating history:
./bench.sh --no-history --output results.json
```

## Size Limit

History is limited to the 100 most recent runs to keep file size manageable.
Older results are automatically pruned by `scripts/append-benchmark.py`.
